# Organization

- Global settings
- Presets:
    - 24 presets saved in memory: 
        - banks A..D
        - presets 1..6 within each bank
    - Preset #0 is current preset (in volatile memory)
        
        
## Preset

- Switches:
    - stompswitch A..D    
    - stompswitch 1..6
    - ext. footswitch 1..4
    - ext. expression pedal 1..2

        
For a switch:

- 6 steps
- control mode
- LEDs settings
    - on color
    - off color
    - MIDI control


For a step:

- channel
- message type
- data, 3 bytes
- other
